Aplicația trebuie să fie un program CLI (Command Line Interface) interactiv de tip REPL (Read-Eval-Print Loop).
Funcționalități necesare:
1. `add <task>` - Crearea unei sarcini noi.
2. `done <id>` - Marcarea unei sarcini ca fiind rezolvată.
3. `delete <id>` - Ștergerea unei sarcini.
4. `list` - Listarea sarcinilor.
5. `exit` - Ieșirea din aplicație.
